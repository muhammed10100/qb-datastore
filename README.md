# Script İsmi: qb-datastore

> **Kurulum**

> **Dosyayı `resources` ın içine atıp start veriyoruz.**

> **Server.cfg Satır Random**
```cfg
start qb-datastore
```
> **Yada Farklı Bir Yöntem**
```cfg
ensure qb-datastore
```
> **Database tablosunu okutup çalıştıra bilrisinz.**
